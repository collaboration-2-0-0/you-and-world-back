# Мета презентації

Пошук бажаючих прийняти участь у створенні продукту

# Тип проекту

Open source project

# Чим може бути цікавий

1. можливість практичного застосування знань по patterns, local-first
2. можливість менторінга розробників
3. продукт а не учбовий проект, який можна використовувати вже в процесі розробки
4. можливість пошуку замовників, виконавців, менторів або однодумців (по мірі розвитку продукту)

# Форми участі

1. командна (один колектив)
2. суперницький (паралельна розробка кількох команд для розробки оптимального рішенння)
3. особистий (своя розробка на основі поданої ідеї)

# Матеріали

можна взяти за основу для нового проекту 'з нуля' (шо я і планую), або дописувати (рефакторти) існуючий
